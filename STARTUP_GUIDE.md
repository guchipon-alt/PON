# 🚀 Руководство по запуску ПОН

## ✅ Требования

- **Node.js** версии 18 или выше
- **npm** или **yarn**
- Минимум 500MB свободного места

---

## 📋 Шаг 1: Подготовка

### Клонирование репозитория
```bash
git clone https://github.com/guchipon-alt/PON.git
cd PON
```

### Установка зависимостей
```bash
npm install
```

Это установит все необходимые пакеты (React, Express, Tailwind, Gemini API, и т.д.)

---

## 🔐 Шаг 2: Конфигурация

### Создание файла .env.local

В корне проекта создай файл `.env.local`:

```bash
cp .env.example .env.local
```

**Минимальная конфигурация (все работает без этого):**
```
PORT=3000
NODE_ENV=development
```

**Для полного функционала с ИИ (опционально):**
```
GEMINI_API_KEY=your_api_key_here
PORT=3000
NODE_ENV=development
```

**Получить Gemini API key:**
1. Перейди на https://ai.google.dev/
2. Нажми "Get API key"
3. Создай ключ для своего проекта
4. Скопируй ключ в `.env.local`

---

## 🎯 Шаг 3: Запуск приложения

### Режим разработки (с HMR - живая перезагрузка)
```bash
npm run dev
```

Приложение запустится на:
- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:3000

Открой в браузере одну из этих ссылок.

### Production режим
```bash
npm run build      # Собрать приложение
npm run start      # Запустить собранную версию
```

## 📝 Шаг 4: Первый запуск

### Вход в приложение

При первом запуске появится экран регистрации. Ты можешь либо:

1. Нажми "Создать аккаунт"
2. Введи email и пароль
3. Введи код подтверждения (в demo режиме показывается в консоли)
4. Готово! 🎉

#### Вариант 2: Использовать демо аккаунт
- **Email**: pon@gmail.com
- **Пароль**: 1234
- Просто введи эти данные и нажми "Войти"

#### Вариант 3: Админ аккаунт
Используй юзернейм **@guchipon** при регистрации для получения админ доступа.

## 🎮 Тестирование функций

### Создание поста
1. Нажми **"Опубликовать"** или кнопку **"+"** на мобильном
2. Напиши текст
3. (Опционально) Добавь изображение
4. Нажми **"Опубликовать"**

Пост будет проверен ИИ в фоне и отобразится на ленте.

### Лайки и комментарии
- ❤️ Нажми на сердце для лайка
- 💬 Нажми на пузырь речи для комментариев
- Поделись постом или добавь в закладки

### Поиск
1. Перейди в раздел **"Поиск"**
2. Введи текст, имя автора или хэштег
3. Поиск работает в реальном времени

### Профиль
1. Перейди в **"Профиль"**
2. Нажми **"Редактировать"**
3. Измени аватар, биографию
4. Рисуй баннер кистью прямо в браузере!

### Поддержка
1. Нажми **"Поддержка"** (если видишь)
2. Задай вопрос техподдержке
3. Получи ответ от ИИ ассистента

## 🐛 Решение проблем

### Приложение не запускается

**Ошибка: "Port 3000 is already in use"**
```bash
# Измени порт в .env.local
PORT=3001
```

**Ошибка: "Cannot find module"**
```bash
# Переустанови зависимости
rm -rf node_modules package-lock.json
npm install
```

### Приложение зависает при запуске

1. Проверь интернет соединение
2. Убедись что не превышена квота Gemini API (если используешь)
3. Перезагрузи терминал и попробуй снова

### Посты не создаются

- Проверь консоль браузера (F12 → Console) на ошибки
- Если видишь "AI Moderation rejected" — текст содержит неприемлемый контент
- В demo режиме все работает локально, API ключ не требуется

### Личные сообщения не работают

Для DM нужно иметь взаимные подписки. В demo режиме это автоматически включено для тестовых аккаунтов.
